# Fiesta Platter AI Rules

Before generating any Next.js code:

- Read official Next.js documentation.
- Prefer Server Components.
- Avoid Client Components.
- Optimize bundle size.
- Optimize images.
- Use Route Groups.
- Use Metadata API.
- Use Server Actions.
- Use App Router only.
- Follow accessibility guidelines.
- Keep components reusable.
- Mobile first.
- SEO first.
